## Star-LED-Chaser
Hi! I am Maritza and in this project I made a 555 LED Chaser with a PCB that look like a star.
I wanted to make it as a keychain

# Feature
- NE555P (555 TImer)
- CD4017 (Integrated Circuit)
- 10 LEDs
- 2 Capacitor
- 2 Resistor
- 1 Variable Resistor
